Use jQuery to expand your Pardot forms, quite literally

Yes, that's jQuery. A very old but powerful JavaScript library!
jQuery has been around since 2006 and has seen plenty of releases ever since.

Did you know that you can use jQuery with Pardot to add interactivity to your forms?
Well, it turns out that you can, and it's as easy as 1 2 3. We'll show you how!

The first thing you need to do to produce the effect that you just saw is adding CSS classes to the fields in your form.

A CSS class is a name you that give to either one or many elements in your form to reference them and add some styling to them.

For this particular use case, we'll add a class that hides the fields that you don't want to display upon loading the page.

Step 1: Go to your form and edit the fields that you would like to remain hidden. You can find the CSS class field in the Advanced tab after you click edit. We'll name our class 'hide'.

Step 2: Edit the Layout Template on your form and add the following code snippet to it. This will be the styling and will take care of hiding the fields for you. Do not worry if there's anything else in there, just make sure to include the snippet between the opening and closing head tags.

                
    <style>
        .hide {
            display: none;
        }
    </style>

Step 3: You'll need to add yet another snippet in but this one needs to be pasted into your form. Head over to the form you want to edit and click the Below Form tab, in the 'Look and Feel' edit section. Then, click the Source icon sitting on the right of the WYSIWYG editor and paste the contents below on the box at the bottom of the edit section.

                                       
    <!-- Call the jQuery lib -->
    <script src="https://code.jquery.com/jquery-3.6.0.js"></script>
    <!-- This is the actual function -->
    <script>
        $(function() {
            $( "p input" ).keydown(function() {
                $( ".hide" ).show();
            });
        });
    </script>

And there you have it, one easy fix that makes your forms look 10 times cooler. 😎
